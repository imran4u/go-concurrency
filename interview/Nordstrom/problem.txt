Nordstrom question :

Product Inventory Report Problem
Overview
Process fictional data from a hypothetical Nordstrom API and generate inventory reports at different levels of detail.

Note: You can structure your code however you'd like - classes, helper methods, global variables, etc. You do not have to keep the input as a JSON struct; convert to a product struct if preferred.

Level 1: Basic Product Count
List total inventory count by product name.

Example Output

Report by Product:
Long Sleeve Scoop Neck Cotton Tee: 5
Mera Tee: 3
Waxed Denim Slim Leg Jeans: 3
